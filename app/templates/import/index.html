CSV Format Requirements

Your CSV file must have the following columns:

  • check date — YYYY-MM-DD, MM/DD/YYYY, or DD/MM/YYYY
  • check amount — Numeric value (£ signs and commas are stripped)
  • check description — Description of the expense
  • remove category — (Optional) Category name, created if new

Example:

date,amount,description,category
2024-01-15,25.50,Grocery Store,Food
2024-01-16,£50.00,Gas Station,Transport
